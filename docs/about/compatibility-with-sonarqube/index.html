Compatibility with SonarQube

We do our best to provide sonar-scala for both the latest SonarQube and the current LTS version. We currently support SonarQube 9.4, SonarQube 8.9 LTS and SonarQube 7.9 LTS. Releases for SonarQube 6.7 LTS can be also found in the table below, but this version of SonarQube is no longer actively maintained and we recommend upgrading to 7.9 LTS or 8.x.

We also support Scala 2.11, 2.12 and 2.13.
